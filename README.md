### ACTIVIDADES 

Cree una aplicación PWA que mediante el uso de promesas simuladas se asegure que procesa el pago de una compra realizada en una app web de e-commerce. Para esto tiene que crear una interfaz web que permita obtener los datos del usuario (Nombre, cédula, número de tarjeta de crédito, fecha expiración, CVV, etc).:
* Cuando la compra se registra exitosamente
* Cuando la compra no se puede realizar porque no hay conexión a internet
* Cuando la compra no se puede realizar porque los datos de la tarjeta de crédito son incorrectos.

Implemente esta aplicación haciendo uso de la mayor cantidad de métodos del objeto Promise en JavaScript.
